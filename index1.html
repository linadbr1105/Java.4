1. 코드 실행흐름 제어 정상적인 코드 실행 흐름은 main()메소드의 시작 중괄호{에서 끝 중괄호}까지 위에서 아래방향으로 실행된다. 그런데 제어문은 정상적인 코드 실행 흐름을 개발자가 원하는 방향으로 변경할 수 있도록 해준다. 그 제어문의 구성 요소로는 조건식과 중괄호가 있다. 조건식은 중괄호 내부를 실행할지를 결정하고 중괄호는 제어문의 실행 내용을 담는다. 제어문의 종류에는 조건문, 반복문, break문, continue문이 있다. 제어문은 중괄호 내부에 다른 제어문을 작성 가능해서 중첩을 할 수 있다. 2. 조건문 1 if문: 조건식의 결과에 따라 중괄호 블록을 실행할지 여부를 결정할 떄 사용. 조건식에는 true 또는 false값을 산출할 수 있는 연산식 또는 boolean변수가 온다. 조건식이 true이면 블록을 실행하고, false이면 블록을 실행하지 않는다. if-else문 : 조건식이 만족되면 if문을, 만족 안되면 else문을 실행한다. if-else if -else문 : 조건식이 만족되면 if문을, 안되면 else if를, 그것도 만족 안되면 else를 실행한다. 중첩 if문: if문 안에 또 다른 if문을 작성할 수 있다. 2 switch문 : 변수나 연산식의 값에 따라 실행문을 선택할 때 사용한다. 3. 반복문 : 중괄호 블록 내용을 반복적으로 실행할 떄 사용한다. 1 for문 : 반복 횟수를 알고 있을 떄 주로 사용 2 while문 : 조건에 따라 반복을 계속할지를 결정할 때 사용 3 do-while문 : 조건이 실행문 뒤에 나와서 실행문을 한번은 무조건 실행하게 되어있고, 반복할지 말지만 결정. 4 break문 : for문, while문, do-while문을 종료한다. Switch문을 종료한다. 반복문이 중첩되어 있을 경우 : 중첩된 경우 break는 가장 가까운 반복문만 종료. 바깥쪽 반복문까지 종료시키려면 반복문에 이름(Label)을 붙이고, 종료시에는 "break 이름(Label);"을 사용. 5 continue문 : for문, while문, do-while문에서 사용. for문은 증감식으로 이동, While문, do-while문은 조건식으로 이동. 즉, 반복문의 처음으로 다시 돌아간다고 보면 됨.
